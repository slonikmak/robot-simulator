# Симулятор поведения робота на выставке

## Концепция


Робот моделирует поведение биологического организма, ищущего место для рождения потомства.

## Техническая реализация

Робот оснащён:
- 2 ведущими колёсами для передвижения
- 3 ToF.
- микроконтроллером для управления поведением робота
- Механиз метания икры (например, небольшая катапульта или шприц). Для симуляции не важно, что именно используется, главное  визуальный эффект от метания икры.
  
## Что нужно сделать

Нужна симуляция поведения робота, похожая на описанное выше. Используем html, css и js для создания визуальной модели робота и его поведения.
Вид сверху, где видно зону обитания и робота. Робот должен двигаться, реагировать на появление "ног" (можно симулировать с помощью курсора мыши) и выполнять описанные действия (метать икру, двигаться к ногам).
Важно чтобы поведение робота соответствовало возможностям реального робота с учётом технических ограничений, то есть не должно быть слишком сложных движений или реакций, которые сложно реализовать на практике при помощи существующих компонентов.

Важно, чтобы симуляция была достаточно простой для понимания, но при этом демонстрировала основные аспекты поведения робота, описанные в концепции. 

Это нужно для проверки концепции и понимания того, как робот будет вести себя в реальной жизни, а также для выявления возможных проблем или улучшений в дизайне робота. 

Если что-то не реализуется с помощью техники робота то важно отразить реальные ограничения в симуляции, чтобы не создавать нереалистичных ожиданий относительно возможностей робота.

Робот размером 30 см в диаметре. Вид сверху, пусть робот выглядит как божья коровка с прозрачным корпусом с икринками, а зона обитания - это круг с разметкой на полу.

Вид можно приближать/удалять. 

Код поведения робота - близкий к тому что можно сделать на arduino, и максимально точная симуляция датчиков и моторов. Метание икры - это просто визуальный эффект, который можно реализовать с помощью анимации.
